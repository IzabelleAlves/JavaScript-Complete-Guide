### Operadores Aritméticos

| Operador | Descrição                 | Exemplo          | Resultado |
| -------- | ------------------------- | ---------------- | --------- |
| `+`      | Adição                    | `5 + 3`          | `8`       |
| `-`      | Subtração                 | `5 - 3`          | `2`       |
| `*`      | Multiplicação             | `5 * 3`          | `15`      |
| `/`      | Divisão                   | `6 / 3`          | `2`       |
| `%`      | Módulo (resto da divisão) | `5 % 3`          | `2`       |
| `**`     | Exponenciação (potência)  | `5 ** 2`         | `25`      |
| `++`     | Incremento                | `let a = 5; a++` | `6`       |
| `--`     | Decremento                | `let a = 5; a--` | `4`       |
